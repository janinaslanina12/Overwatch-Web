About the game

Welcome to the Virtual Plaza

Vaporwave je přeci jenom o poslechu a hudbě jako takové. Zde máte pár mých oblíbených skladeb.
Avšak než vám popřeju příjemné poslouchání, chtěl bych poděkovat jednomu fanouškovi vystupujicím pod jménem System ST91, který mi před rokem napsal, abych si vyslechl jeho album. To, že by můj skromný web někoho inspiroval k vlastní tvorbě mě nesmírně dojalo a chtěl bych se System ST91 osobně omluvit, že jsem nereagoval dříve. S hrdostí zde prezentuji jeho album jako příklad českého vaporwave.
A nyní s chutí do toho!

- Dave B.
